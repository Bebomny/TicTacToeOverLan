Player ServerUtils::clientContextToPlayer ( const ClientContext & client,
bool requestingPlayerId )
static

Converts the server-side internal client representation into a public Player struct.
Used for network synchronization, as ClientContext contains private data that should not be sent to clients, whereas Player is the "public" view of a user suitable for the UI.

Parameters
clientThe internal client context to convert.
requestingPlayerIdThe ID of the player requesting this data (or the ID of the client to whom this packet will be sent). This is used to correctly set the isMe flag.
Returns
A Player struct populated with safe-to-share data.
