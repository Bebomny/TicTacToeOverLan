Determines if the last move resulted in a win.
Scans four axes (Horizontal, Vertical, Diagonal /, Diagonal ) centered on the last move.
Logic: For each axis, it counts identical pieces in the positive direction and negative direction.
If (count1 + count2 + 1) >= board.winConditionLength, the player has won.

Parameters
boardThe current state of the game board.
lastXThe X coordinate of the most recently placed piece.
lastYThe Y coordinate of the most recently placed piece.
Returns
True if the move created a winning line, false otherwise.
