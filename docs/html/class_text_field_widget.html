Processes input for the text field.
Handles two main things:
1. Mouse Clicks: Checks if the user clicked inside (gain focus) or outside (lose focus).
2. Text Input: If focused, handles sf::Event::TextEntered to update the string.

Parameters
eventThe system event (required for character typing).
mousePosCurrent mouse position.
Returns
True if the event was consumed (e.g., user typed or clicked the field).
