\doxysubsection{Detailed Description}
Abstract base class representing a UI element. 


\begin{DoxyItemize}
\item The \doxylink{class_widget}{Widget} class provides the foundational interface for all UI components, managing visibility states, activity states, and the core game loop methods (event handling, updating, and rendering). 
\end{DoxyItemize}
